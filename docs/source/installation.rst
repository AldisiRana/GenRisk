Installation
##############

Requirements
_____________
* `plink <https://www.cog-genomics.org/plink>`_ >= 1.9
* R version >= 3.6.3
* python >= 3.

Package installation
____________________
``GenRisk`` can be installed on python3+ from the latest code on `GitHub <https://github.com/AldisiRana/GenRisk>`_ with:

.. code-block:: bash

    pip install genrisk
